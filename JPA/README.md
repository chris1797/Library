# JPA Programming Study Notes

이 디렉토리는 JPA(Java Persistence API) 프로그래밍을 공부하면서 정리한 내용과 개인적인 메모를 저장하기 위해 만들어졌다.

## 주요 내용

### - JPA 기초
- 엔티티 매핑(Entity Mapping)
- 기본적인 CRUD 연산, 기능 및 Annotation

### - 고급 JPA
- JPQL(Java Persistence Query Language)
- 고급 매핑(상속, 조인 전략 등)
- 성능 최적화
- 트랜잭션 관리
